# Typography
There are some predefined heading and text sizes and styles. They should only be extended after coordination with the ui/ux responsible
## Heading
Headings can be defined using `.heading` classes or native `h`-Tags

### Sizes
The predefined Sizes can be changed globally when needed. It is recommended to not add new sizes but to scrutinise the ui if this is really necessary.
